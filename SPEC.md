# SPEC — Idea Capture App

## App
Idea Capture is a tiny CLI-first app to quickly capture ideas with tags and list/search them later.

## Target user
Solo builders and creators who need a frictionless way to save ideas from terminal workflows.

## MVP
- Add an idea with text and optional comma-separated tags.
- Persist ideas in a local JSON file.
- List all ideas newest-first.
- Filter ideas by tag.
- Basic input validation and clear CLI feedback.
- Minimal automated test coverage for core flows.
